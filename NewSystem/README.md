# NewSystem — Starter scaffold

This is a minimal starter scaffold for a new system combining a FastAPI backend and a Streamlit frontend.

Run locally:

```powershell
cd "C:\xampp\htdocs\GenAIClass\TheFutureOfGenAiClass\NewSystem"
call START_SYSTEM.bat
```

Files:
- `backend/app.py` — FastAPI app with `/health`.
- `frontend/streamlit_app.py` — Streamlit UI that checks backend health.
- `requirements.txt` — minimal dependencies.
- `START_SYSTEM.bat` — quick Windows launcher to create a venv and run both services.
