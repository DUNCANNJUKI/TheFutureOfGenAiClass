## üöÄ How to Use

### Windows (Quick Start)
```bash
1. Double-click START_SYSTEM.bat
2. Wait for "Streamlit app is now running"
3. Browser opens to http://localhost:8502
```

### Linux/Mac
```bash
1. python -m venv venv
2. source venv/bin/activate
3. pip install -r CodebaseGenius/FE/requirements.txt
4. streamlit run CodebaseGenius/FE/code_master.py --server.port 8502
5. Open http://localhost:8502
```

### Manual Setup
```bash
1. Review SETUP_GUIDE.md for detailed instructions
2. Configure .env if needed
3. Follow step-by-step installation
4. Run START_SYSTEM.bat or manual commands
```
